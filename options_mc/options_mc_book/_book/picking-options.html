Buying Derivatives - A Monte Carlo Approach

3 Picking Options

Using the simulation outlined on the previous page, I then evaluate options from the companies with the highest market capitalization (definition).

  1. Simulate the expected values for all available strike prices (calls and puts) for a given options expiration date for all stocks on the list
  2. Remove any stock options that do not have a positive expected value
  3. Create a few metrics to evaluate each option
  4. Pick the “best” option for each stock with a positive expected value
  5. Evaluate the cumulative list

I want the most bang for my buck, though. So I created a metric to find out which options had the biggest differential between cost and expected value.

dis_puts %>% 
    mutate(exp_spent = ifelse(type == "call", 
                              ask * 100 * `Llhd Blw EP`,
                              ask * 100 * (1 - `Llhd Blw EP`)),
           exp_made = ifelse(type == "call", 
                             (`EV Abv EP` - `EP Buy`) * (1 - `Llhd Blw EP`) * -100,
                             (`EV Blw EP` - `EP Buy`) * `Llhd Blw EP` * -100),
           width = exp_made - exp_spent,
           wperc = width / exp_spent) %>% 
    group_by(symbol) %>% 
    slice_max(wperc, n = 1) %>% 
    ungroup() %>% 
    arrange(desc(wperc)) %>% 
    head(n = 10) %>% 
    mutate(CP = dollar(CP),
           strike = dollar(strike),
           ask = dollar(ask),
           `EP Buy` = dollar(`EP Buy`),
           exp_spent = dollar(exp_spent),
           exp_made = dollar(exp_made),
           wperc = percent(wperc)) %>% 
    select(symbol,
           CP,
           type,
           strike,
           ask,
           `EP Buy`,
           exp_spent,
           exp_made,
           wperc)%>% 
    kbl() %>%
    kable_styling(bootstrap_options = c("striped", "hover"))
